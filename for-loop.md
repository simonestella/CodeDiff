# For loop
Here is explained the syntax of the *for loop*.

## Python
**Important!** In Python indentation is really important.
```python
for x in range(5):
  # Do something
```

## Perl
```perl
if (a > b) {
  # Do something
} elsif (a < b) {
  # Do something
} else {
  # Do something
}
```

## Java
```java
// Classic for loop
for (int i = 0; i < 5; i++) { 
    // Do something
}
    
// Enhanced for loop
for (int item : items) {
  // Do something
}
```

## C++
```cpp
if (a > b) {
  // Do something
} else if (a < b) {
  // Do something
} else {
  // Do something
}
```
