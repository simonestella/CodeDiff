# If statement
*if statement* used to specify a block of code to be executed, if a specified condition is true.

## Python
**Important!** In Python indentation is really important.
```python
if a > b:
  # Do something
elif a < b:
  # Do something
else:
  # Do something
```

## Perl
```perl
if (a > b) {
  # Do something
} elsif (a < b) {
  # Do something
} else {
  # Do something
}
```

## Java
```java
if (a > b) {
 // Do something
} else if (a < b) {
 // Do something
} else {
 // Do something
}
```

## C++
```cpp

  // Do something

```
